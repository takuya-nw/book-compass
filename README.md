# Book Compass

Book Compassは、個人向けの読書管理・本棚Webアプリです。Google Booksでの本の検索、表紙付き検索結果、詳細確認、本棚登録、読書ステータス管理、読み始めた日・読み終えた日の編集、自分用の読後評価・メモ、ホーム画面の読書集計、JSONバックアップに対応しています。

## 必要環境

- Node.js 20以上
- npm

## インストール方法

```bash
npm install
```

## 起動方法

```bash
npm run dev
```

起動後、ブラウザで `http://localhost:3000` を開きます。

## APIキーの設定方法

`.env.example` を参考に `.env.local` を作成し、必要なキーを設定してください。

```bash
GOOGLE_BOOKS_API_KEY=
```

APIキーはブラウザ側へ直接渡さず、Next.jsのAPI RouteからGoogle Books APIへアクセスします。外部サービスの登録が必要な場合は、ユーザー自身で登録してください。

楽天ブックスAPIは当面使用しない方針です。将来追加しやすいようにサービス層は画面から分離していますが、現在の検索画面ではGoogle Booksのみを使います。

## デモモードの使い方

APIキーが未設定の場合でも画面確認できるよう、検索APIはサンプルデータを返します。`.env.local` を作らない、またはキーを空欄にした状態で検索するとデモ用の本が表示されます。

## データの保存場所

本棚データはブラウザの `localStorage` に保存されます。初期実装ではクラウド保存を行いません。データアクセス処理は `repositories` に分離しているため、後からSupabaseなどへ移行しやすい構造です。

## バックアップ方法

マイ本棚画面の「JSONでエクスポート」からバックアップできます。「JSONから復元」を使うと、現在の本棚データをバックアップ内容で上書きします。復元前には確認メッセージを表示します。復元データは、各項目の型、日付、ID重複、本と読書記録の対応を確認し、すべて正しい場合だけ保存します。不正なJSONは読み込まず、現在のデータを壊しません。

## 開発時の確認

```bash

npm test
npm run build
```
